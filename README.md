🏠 總管家包租代管 - 房屋資訊管理系統
本系統由 宜仁居家有限公司 技術支援，專為「總管家包租代管」業務同仁設計，用於即時登記房屋物件、追蹤案件進度並自動統計營運數據。
🚀 系統核心功能
•	即時狀態看板：自動統計「物件總數」、「已成交」及「待租中」數量，營運狀況一目了然。
•	數位化登記：包含業務人員、所有權人、地址、租金、管理費、車位、房型及建物型態等完整欄位。
•	進度追蹤：快速切換案件狀態（待租、已成交），資料同步更新。
•	權限控管：嚴格限制存取對象，僅限特定授權帳號進入系統操作。
•	雲端同步：整合 Firebase Firestore 資料庫，多裝置同步，資料不遺失。

🛠️ 技術架構
•	前端：HTML5, Tailwind CSS (響應式設計，支援手機瀏覽)。
•	後端/資料庫：Firebase Firestore (即時雲端資料庫)。
•	部署平台：GitHub Pages。
 
📱 手機快速使用教學
為了讓業務同仁在外面也能快速登記，建議將此網頁加到手機桌面：
iPhone (Safari)
1.	使用 Safari 開啟您的系統網址。
2.	點擊下方的「分享」圖示（正方形箭頭向上）。
3.	選擇「加入主畫面」。
Android (Chrome)
1.	使用 Chrome 開啟您的系統網址。
2.	點擊右上角「三個點」圖示。
3.	選擇「安裝應用程式」或「加到主畫面」。
 
🛡️ 安全維護
•	域名白名單：僅授權於指定的 GitHub 網域及 localhost 執行。
•	資料安全：建議定期於 Firebase 控制台檢查資料備份。
